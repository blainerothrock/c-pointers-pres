# Pointers in C - Lecture Presentation

An interactive [Slidev](https://sli.dev/) presentation covering pointers in the C programming language. Includes live-editable and runnable C code examples via Monaco editor.

## Topics

1. Program memory layout (stack, heap, data, text segments)
2. Conceptualizing pointers
3. Pointer syntax (declaration, dereferencing, address-of)
4. Pass-by-value vs pass-by-reference
5. Pointer arithmetic and arrays

## Development

```bash
pnpm install
pnpm dev
```

Open <http://localhost:3030> to view the slides.
